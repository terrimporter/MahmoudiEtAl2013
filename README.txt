Supporting files for the Mahmoudi et al., 2013 publication:

Mahmoudi, N., Porter, T.M., Zimmerman, A.R., Fulthorpe, R.R., Kasozi, G.N., Silliman, B.R., and Slater, G.F. (2013) Rapid Degradation of Deepwater Horizon Spilled Oil by Indigenous Microbial Communities in Louisiana Saltmarsh Sediments.  Environmental Science & Technology, 47: 13303-13312.

Text/Fig4/Archaea - contains the OTU counts used to create stacked bar charts
Text/Fig4/Bacteria - contains the OTU counts used to create stacked bar charts
SuppMat/FigS3/Archaea_NMDS - contains csv files and an R script to create NMDS plots
SuppMat/FigS3/Bacteria_NMDS - contains csv files and an R script to create NMDS plots
SuppMat/FigS3/Fungi_NMDS - contains csv files and an R script to create NMDS plots
